❓ Database won't deploy

   └─ Check browser console (F12), verify RESEND_API_KEY is set

❓ Pages don't load
   └─ Check npm run dev logs, verify no syntax errors

❓ Styling looks wrong
   └─ Hard refresh (Cmd+Shift+R), verify Tailwind classes applied

❓ Emails not arriving
   └─ Check Supabase table (was data inserted?), check Resend logs
